# A.I. Pathfinding & Swag Route Finder

Take complete command of geometry routing with this dynamic visual Pathfinding Mod for Geometry Dash 2.2.

### How It Works
1. **Obstacle Detection:** The assistant scans nearby physical coordinates in front of your icons.
2. **Ground & Air Jumps:** When a spike or block hazard is near, the assistant automatically issues a jump pulse.
3. **Orb Routing Logic:** When approaching custom rings/orbs, it handles micro-clicks safely.
4. **Find Swag Routes:** By disabling "Orb Detection" in the settings, the player bypasses the developer's default setup paths! Use this in **Polargeist** to skip the classic yellow rings and drop directly into the hidden bottom route to secure the elusive coins.

### Visuals Mode
Features a real-time drawing layer displaying predictive safety paths (green lines for successful orb triggers, red warnings for incoming jump hazards).
